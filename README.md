## Features

- ✅ **Live invoice preview** — See changes as you type, every keystroke
- ✅ **Dynamic line items** — Add, remove, and edit invoice rows in seconds
- ✅ **Smart tax calculation** — Toggle GST/tax with instant auto-computation
- ✅ **Automatic calculations** — Subtotal, discounts, tax, and total computed in real-time
- ✅ **One-click PDF export** — Download professional invoices with jsPDF + html2canvas
- ✅ **Fully responsive** — Works seamlessly on desktop, tablet, and mobile
- ✅ **Zero friction** — No login, no signup, no data collection — everything client-side
- ✅ **Dark UI** — Clean, modern interface built for freelancers who mean business

## Getting Started

### Prerequisites

- Node.js 18+ and npm/yarn/pnpm installed

### Clone & Install

```bash
git clone https://github.com/Muzzu8421/freelance-invoice-generator.git
cd freelance-invoice-generator
npm install
```

### Run Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser. Hot reload is enabled.

### Build for Production

```bash
npm run build
npm start
```

## Usage

1. **Enter your details** — Add your name, email, phone, and business info
2. **Add client info** — Fill in the client name, email, and address
3. **Create line items** — Add services/products with quantity and rate
4. **Toggle tax** — Enable/disable GST with one click
5. **Set discount (optional)** — Apply a percentage or fixed discount
6. **Download PDF** — Hit the export button and get your invoice instantly

All calculations happen locally. Your data never leaves your browser.
